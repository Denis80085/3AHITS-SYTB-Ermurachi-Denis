## Übung (Directory Struktur)

**Schreibe ein Shell-Script das Unterverzeichnisse und Dateien anlegt, es wird ein Argument an das Script übergeben.**

**Aufruf:** ```./build_dirs.sh abcd```

**Dies führt zu folgender Directorystruktur:**
```
./
└── abcd/
    ├── abcd_01/
    │   ├── abcd.01.1.txt
    │   ├── abcd.01.2.txt
    │   └── abcd.01.3.txt
    └── abcd_02/
        ├── abcd.02.1.txt
        ├── abcd.02.2.txt
        └── abcd.02.3.txt
```

**Lösung:**

> _Prompt: Erkläre wie die Argumenten in Bashscript funktionieren. Wie kann man eine Verzeichniss in einer Variable speichern und mit einer Benutzereingabe erweitern, sodass eine neue Verzeichniss generiert werden kann._ 
```bash
#!/bin/env bash
set -euo pipefail

name="${1:-}"
if [[ -z "$name" ]]; then
    echo "Usage: $0 dirname"
    exit 1
fi

root="$(pwd)/$name"
mkdir -p "$root"

for i in {1..2}; do
    subdir="${name}_0${i}"
    mkdir -p "$root/$subdir"
    for j in {1..3}; do
        touch "$root/$subdir/${name}.0${i}.${j}.txt"
    done
done

tree "$root"
```

**Schreibe weiters ein Shell-Script ```clean_dir.sh``` das diese Verzeichnisstruktur wieder löscht.**

**Hinweis: Das Kommando ```tree``` kann verwendet werden um eine Directory Struktur in obiger Form darzustellen.**

**Lösung:**
```bash
#!/bin/env bash
set -euo pipefail

name="${1:-}"
if [[ -z "$name" ]]; then
    echo "Usage: $0 dirname"
    exit 1
fi

root="$(pwd)/$name"
rm -rf "$root"
```
