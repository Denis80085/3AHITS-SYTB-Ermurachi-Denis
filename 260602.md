#### Übung (MAC-Adressen validieren)

Schreibe einen ERE-Ausdruck für MAC-Adressen.

Format:

```text
AA:BB:CC:DD:EE:FF
```

Regeln:

- genau 6 Gruppen
- jeweils 2 hexadezimale Zeichen
- Trennung durch `:`

#### Lösung

```
([a-fA-F0-9][a-fA-F0-9]:){5}[a-fA-F0-9][a-fA-F0-9]
```

#### Übung (MAC-Adressen validieren - Skript)

Erstelle ein shell script das eine Mac Adresse als Argument übergeben bekommt und diese mit Hilfe eines ERE-Ausdrucks validiert. Es soll eine passende Ausgabe erzeugt und der exit Status gesetzt werden, so dass man dieses Script aus einem anderen Script heraus aufrufen kann.

#### Script

```bash
#!/bin/env bash

HEX_IN=$1

REGEX="([a-fA-F0-9][a-fA-F0-9]:){5}[a-fA-F0-9][a-fA-F0-9]"

if [[ $HEX_IN =~ $REGEX ]] 
then
    printf $HEX_IN | grep -E $REGEX
else 
    echo "Invalid Hex"
fi
```

#### Übung (Funktionsdefinitionen in C finden)

Finde einfache C-Funktionsdefinitionen.

Beispiele:

```c
int add(int a, int b)
void test()
char* name()
```

Regeln:

- Rückgabetyp besteht aus Buchstaben und optional `*`
- Funktionsname besteht aus Buchstaben/Ziffern/`_`
- Klammern müssen vorhanden sein

#### Lösung

```
(void|char|int|bool|float)[*&]?\s\w.+\(.*\)
```

#### Übung (Funktionsdefinitionen in Python)

Lasse dir von genAI erklären wie man eine Datei deren Pfad als Argument an das Python Skript übergeben wurde in einem Python Skript einliest und Zeile für Zeile auf der Konsole ausgibt.

Recherchiere weiters wie man in Python Regular Expressions anwenden kann um Funktionsdefinitionen in einem C-Programm zu finden.

Das Python Skript soll dann alle gefundenen Funktionsdefinitionen auf der Konsole ausgeben.

Arbeite schrittweise von einer lauffähigen Ausbaustufe des Skripts zur nächsten, lasse nicht gleich die fertige Lösung von der KI generieren.

#### Regex

Expression von Stack Overflow [Link](https://stackoverflow.com/questions/943391/how-to-get-the-function-declaration-or-definitions-using-regex)
```
^\s*(?:(?:inline|static)\s+){0,2}(?!else|typedef|return)\w+\s+\*?\s*(\w+)\s*\([^)]*\)\s*\{
```

#### Phython Code

```Das Script wurde gemacht mit der Hilfe von Stack Overflow und KI```
[Stack Overflow Artikel](https://stackoverflow.com/questions/943391/how-to-get-the-function-declaration-or-definitions-using-regex) 

```phython
import sys
import re

def getFunctionDefs(zeile):
    rproc = r"((?<=[\s:~])(\w+)\s*\(([\w\s,<>\[\].=&':/*]*?)\)\s*(const)?\s*(?={))"
    cppwords = ['if', 'while', 'do', 'for', 'switch']
    proc = [(i.group(2), i.group(3)) for i in re.finditer(rproc, zeile) \
    if i.group(2) not in cppwords]
    for i in proc:
        print(i[0] + '(' + i[1] + ')')

def main():
    if len(sys.argv) < 2:
        print("Bitte einen Dateipfad als Argument angeben.")
        return

    dateipfad = sys.argv[1]

    with open(dateipfad, "r", encoding="utf-8") as f:
        getFunctionDefs(f.read())  # end="" verhindert doppelte Zeilenumbrüche


if __name__ == "__main__":
    main()
```
