#### Übung (Even/Odd)

Create a simple script which will print the numbers 1-42 (each on a separate line) and whether they are even or odd.

#### Skript

```bash
#!/bin/env bash

for n in {1..42}
do
    if (($n % 2 == 0))
    then
        echo "$n even"
    else
        echo "$n odd"
    fi
done
```

#### Übung (Summe und Maximum)

Schreibe ein bash Skript das eine **beliebige** Menge von positiven Zahlen als Argumente aus der Kommandozeile übernimmt. Es soll die Summe und die größte Zahl ausgegeben werden. Hinweis: Verwende `#@`.

#### Skript

```bash
#!/bin/env bash

sum=0

for arg in $@
do
    ((sum+=$arg))
done

echo "summe: $sum"
```

#### Übung (Min)

Schreibe ein bash Skript das den Namen einer Datei als Argument akzeptiert:

```bash
$ ./min.sh input.txt
```

Die Input Datei enthält positive Zahlen die durch white spaces getrennt sind:

```
42
21 57
68 33 1
1

```

Das Skript soll die kleinste Zahl ermitteln und ausgeben.

Hinweis:

Der größte mögliche Wert in bash scripting ist 64 Bit signed

```sh
$ ((MAXINT=2**63-1))
$ echo $MAXINT
9223372036854775807
```

#### Skript
```bash
#!/bin/env bash

((CURRENTMIN=2**63-1))
inhalt=$(cat $1)

for n in $inhalt
do
    if (($n < $CURRENTMIN))
    then
        CURRENTMIN=$n
    fi
done

echo "min $CURRENTMIN"
```

**Befehl:** ```./min.sh in.txt```

#### Übung (Durchschnitt)

Schreibe ein bash Skript das eine beliebige Anzahl von Zahlen von **stdin** akzeptiert und den (ganzzahligen) Durchschnitt der Zahlen ausgibt.

```sh
$ echo 3 5 7 2 4 | ./avg.sh
```

Wie Daten von stdin gelesen werden können ist zu recherchieren.

Teste das Skript und verwende dieses dann um den Durchschnitts-Wert für die Spalte Stück in der Datei [lager.csv](https://www.franzmatejka.at/htl/doc/SYTB_3/testdata/lager.csv) zu finden. Hinweis: Verwende `cut` und `tail` für die vorverarbeitung der Daten.

#### avg.sh
```bash
#!/bin/env bash

STDIN=$(cat -)
CONTER=0
SUM=0

for n in $STDIN
do
    ((COUNTER++))
    ((SUM+=$n))
done

avg=$(($SUM/$COUNTER))

echo "avg: $avg"
```

#### Befehl für AVG von Stück

```sh
cat lager.csv | tail -n +2 | cut -d"," -f2 | ./avg.sh
```
