## Übung (Variablen)

*Löse diese Aufgabe ohne ein shell-script zu schreiben.*

*Lege 2 Variablen für Directories an, eine Variable für den Filenamen und eine Variable für die Dateiendung (Extension).*

*Verwende nur die Variablen anstatt der Directory und Filenamen um folgendes zu lösen:*

- *Erzeuge die Directories (das zweite befindet sich in dem ersten).*
- *Lege in dem zweiten Directory eine Text-Datei mit Inhalt an, verwende Filenamen und Extension (getrennt mit .).*
- *Gib den Inhalt der Datei (unter Verwendung der Variablen) aus.*

**Variabeln definieren**
```bash
    $ DIR1=First
    $ DIR2=Second
    $ FILE=text
    $ EXTENSION=txt
```

**Dierectories erzeugen**
```bash
    $ mkdir -p ${DIR1}/${DIR2}
```

**Text Datei einlegen**
```bash
    $ touch ${DIR1}/${DIR2}/${FILE}.${EXTENSION}
```

**Inhalt der Datei eingeben**
```bash
    $ echo "Hallo. Was geht es ab?" > ${DIR1}/${DIR2}/${FILE}.${EXTENSION}
```

**Überprüfen**
```bash
    $ cat ${DIR1}/${DIR2}/${FILE}.${EXTENSION}
    "Hallo. Was geht es ab?"
```

## Übung (Begrüßung)

*Schreibe ein Shell-Script mit dem Namen begruessung.sh, dieses soll:*
1. *den Benutzer nach seinem Namen fragen*
2. *den Namen einlesen und in einer Variable speichern*
3. *danach ausgeben:*
```
    Hallo <NAME>
    Schön, dass du da bist.
```
*Starte das Script auf alle besprochene Arten.*

*Hinweis: Verwende den ```read``` Befehl – recherchiere wie dieser zu verwenden ist.*

Erstens liegt man eine Datei, z.B. script.sh. Hier wird folgender Code geschrieben:

```bash
    #!/bin/env bash

    echo "Welcome! Please enter your name: "

    # Namen auslesen
    read NAME

    # Ausgabe
    printf "Hallo $NAME\nSchön, dass du da bist."
```

```read``` Wartet auf eine Eingabe von Benutzer und Speichert sie in die Variable ```NAME```

## Übung (admin)
*Gegeben ist:*
```
    USER="admin"
```

1. Gib folgende Strings mit echo korrekt aus:
    - admin_backup
    - admin_2026

**Lösung:**
```bash
    echo -e "${USER}_backup"
    echo -e "${USER}_2026"
```

2. Erkläre, warum folgende Zeile nicht das gewünschte Ergebnis liefert:
```
    echo "$USER_backup"
```

**Erklärung:**

In diesem Fall wird nichts ausgegeben, da die Shell versucht, eine Variable namens ```USER_backup``` zu finden, die nicht existiert. Da es keine solche Variable gibt, wird eine leere Zeichenkette ausgegeben.

## Übung (log file names)
*Gegeben ist:* 
```bash
    DIR="/var/log/"
```
*Erzeuge mit echo folgende Ausgaben unter Verwendung der Variabelen DIR:*
```bash
    /var/log/nginx
    /var/log/apache
```

**Lösung:**
```bash
    echo -e "${DIR}nginx"
    echo -e "${DIR}apache"
```

## Übung (backup)
*Gegeben ist:*
```bash
    BASE="backup"
    DATE="2026-01-16"
```
*Erzeuge folgende Dateinamen mit echo:*
```bash
    backup_2026-01-16.tar
    backup_2026-01-16.tar.gz
```
**Lösung:**
```bash
    echo -e "${BASE}_${DATE}.tar"
    echo -e "${BASE}_${DATE}.tar.gz"
```
